# 文本选中复制-通用版
文本选中复制通用版本，适用于大多数网站。  
由于会匹配所有网站，为了减少对于正常网站的干扰，在打开网站之后，**需要自行在网站左下角找到淡蓝色块，鼠标悬浮之后，点击启动 解除复制限制 按钮**。  
某度文库、道客某某等网站需要特殊处理，不能通用处理，请移步 文本选中复制 脚本。
